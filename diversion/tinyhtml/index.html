Forest of Fractal Fluorescent Filaments

mrange here.

I have dabbled in size-coding a bit and watching Lovebyte 2025 I got inspired by the demo Plane parasite to attempt a 1KiB brotli compressed HTML demo myself.

I can create decent visuals but music is a mystery to me so I went to bytebeat composer to look for something I could creatively "borrow". I found a really great bytebeat called Pulsar by luna197 which I tweaked a bit using my poor JavaScript skills to make it fit in 1KiB. I hope I didn't butcher the bytebeat too much while doing so.

Extra big thanks to Pestis for letting me nerd snipe him with some optimizations problems and of course to luna197 for the beutiful music.

The HTML demo when compressed with brotli is less than 1KiB. There are some drawbacks with 1KiB limit, in order to make it start you need to click in the middle of the screen to make it start. This is because Audio playback is prevented without user interaction. Unfortunately with 1KiB limit I don't have space to explain this issue in the screen itself.
