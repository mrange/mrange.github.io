mrange here.

After dabbling in size-coding for a bit, I got inspired while watching the Lovebyte 2025 stream. The demo Plane Parasite really caught my eye and inspired me to try making a 1KiB Brotli-compressed HTML demo.

Visuals are my comfort zone — music, not so much. So I went browsing the wonderful bytebeat composer for something I could, let’s say, “creatively borrow.” I stumbled upon the excellent Pulsar by luna197 and tweaked it (gently!) using some questionable JavaScript to fit it into 1KiB. Hopefully I didn’t mangle it too badly.

Huge thanks to Pestis for letting me nerd-snipe him with optimization puzzles, and of course to luna197 for the beautiful bytebeat.

This HTML demo is under 1KiB when Brotli-compressed, but that tiny size comes with a few trade-offs:

  • User interaction required: You need to click in the middle of the screen to start the demo. This is because browsers block audio playback until there's user input.
  • No on-screen instructions: There's no room to explain this within the demo itself — every byte counts.
  • Fullscreen recommended: For the best experience, put your browser in fullscreen mode (usually with F11).
